# 🔐 Login com Google + Cadastro (React + TypeScript)

## 📌 Descrição do Projeto
Este projeto é uma aplicação web desenvolvida em React com TypeScript, contendo autenticação com Google OAuth.  
Após o login, o usuário pode navegar entre páginas e acessar um formulário de cadastro que pré-preenche automaticamente o nome e e-mail usando os dados do login.

Ao finalizar o cadastro, é gerado um objeto JSON com os dados do usuário, exibido na tela e no console para validação.

---

## 🚀 Funcionalidades
- Login com Google OAuth
- Logout
- Navegação entre páginas (React Router)
- Página "Sobre"
- Página de Cadastro com formulário
- Pré-preenchimento automático de Nome e Email
- Geração e exibição de JSON ao finalizar cadastro

---

## 🛠 Tecnologias Utilizadas
- React
- TypeScript
- React Router DOM
- Google OAuth (`@react-oauth/google`)
- JWT Decode (`jwt-decode`)
- CSS Modules
